# sales-order-automation-uipath

## Overview
This project automates the sales order process from initial creation to final approval using UiPath.

## Features
- Customer validation
- Credit limit checks
- Automated approval workflow
- Excel data processing
- Email notifications

## Technologies Used
- UiPath Studio
- Excel Activities
- Mail Activities

##  Process Flow
1. Read purchase orders from Excel
2. Validate customer and credit data
3. Send for approval
4. Generate final sales order

##  Outcome
- Reduced manual work
- Improved accuracy
